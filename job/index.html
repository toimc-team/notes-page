# 概述

为了帮助小伙伴们更好的求职，toimc开源电子书将试运行【招聘求职】板块。

目前为试运行阶段（截止2022年4月30日），是否持续运营，将根据试运行阶段是否真正帮助到小伙伴们的情况决定。

WARNING

注意：平台内的联系方式仅做招聘求职需要，如出现骚扰、索要金钱、有意泄露他人隐私等问题，请及时反馈给助教或运营处理。

【主要流程】

小伙伴发求职信息到助教/运营企业微信或邮箱 → 助教/运营收集整理到toimc开源电子书 → 助教/运营发群消息到各群 → 助教/运营收集招聘数据

1.小伙伴发求职信息到助教/运营企业微信或邮箱

请小伙伴按格式要求发布招聘求职信息，格式要求如下：

招聘信息格式要求：

【公司】必填

【Base地】必填

【招聘职位】必填

【薪资范围】必填

【用人要求】必填

【简历投递方式】最好是简历接收者的微信二维码，邮箱、手机也可以（如果想保密可填写QQ昵称，通过大前端课程群发起联系）

【附件】建议附上招聘网站带二维码的图片，可提高投递率

注：若多个职位，请发多个求职信息。

求职信息格式要求：

【姓名】想保密昵称也可

【意向城市】

【意向职位】

【行业或领域】

【工作年限】

【联系方式】手机、微信都可

【个人履历】简单介绍下自己的经历和优势（如果想保密可填写QQ昵称，通过大前端课程群发起联系）

【附件】附上简历更高效

注：需要保密的信息可根据自身情况不填。

2.助教/运营收集整理到toimc开源电子书

助教/运营将审核招聘信息提交格式，符合条件将被整理至对外共享文档；不符合条件助教将联系您修改。

3.助教/运营发群消息到各群

助教/运营将在每周二上午发布至大前端课程群（微信、QQ）。

4.助教/运营收集招聘数据

请小伙伴们在投递简历时，备注**“大前端”**，便于用人单位识别以及助教/运营后期收集反馈数据，这也是我们继续做下去的动力。

同时也请招聘者加助教企业微信，备注 招聘-公司名 ，小助手将会在接下来的时间里，每周周一收集一次上周简历投递数据。
