# 概述

互联网发展到今天，安全已经成为用户、企业最为关注的问题之一，前端作为应用的入口，是用户触达应用的门户，也是安全问题频发的高危点。XSS、CSRF、SQL注入、XXE等都是前端常见的安全问题。新型的安全问题不断涌现，浏览器在不断发展抵御这些威胁的时候，我们的前端攻城狮们也需要不断提高自己程序的“安全意识”，来应对这些潜在的安全漏洞。

本章节我们将会梳理出常见的前端安全问题以及对应的解决方案，希望可以帮助小伙伴们在日常的开发中预防和修复这些安全漏洞。
